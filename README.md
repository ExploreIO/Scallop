# 🐚 Scallop

**Scallop** 是一个轻量级、模块化的蜜罐系统，旨在通过模拟真实的服务和网络环境来诱捕攻击者，收集威胁情报并分析攻击行为。

该项目采用模块化设计，目前主要实现了高交互的 SSH 蜜罐，未来将扩展支持 HTTP 蜜罐及可视化内网拓扑蜜罐。

---

## 🌟 项目特性

Scallop 旨在提供逼真的欺骗环境，主要包含以下三大核心功能模块：

### 1. 🛡️ SSH 蜜罐 (开发中)
这是 Scallop 的核心功能。它不仅仅是简单的端口监听，而是一个**全模拟的 Linux 交互环境**。
- **虚拟文件系统**：模拟真实的 Linux 目录结构，让攻击者误以为进入了一台真实的服务器。
- **命令模拟**：支持常用的 Linux 命令（如 `ls`, `cd`, `uname`, `wget` 等），能够根据输入返回伪造的输出。
- **行为画像**：详细记录攻击者的每一步操作，从输入的命令到尝试上传的文件，生成完整的攻击者行为画像。
- **凭证捕获**：安全地记录攻击者尝试使用的用户名和密码组合。

### 2. 🌐 HTTP 蜜罐 (开发中)
*即将推出*
- **页面克隆**：支持根据目标 URL 自动抓取并仿造登录页面或管理后台。
- **模板自定义**：允许用户通过简单的配置或 HTML 模板自定义诱饵页面，用于钓鱼或收集 Web 攻击流量。

### 3. 🕸️ 内网蜜罐 (规划中)
*未来版本*
- **拓扑自定义**：允许用户定义虚拟的内网拓扑结构，模拟复杂的企业网络环境。
- **服务模拟**：在虚拟拓扑中部署多种虚拟服务（如数据库、SMB、Redis 等）。
- **端口伪装**：灵活配置开放端口，迷惑横向移动的攻击者。
